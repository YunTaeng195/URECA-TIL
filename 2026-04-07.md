## 💡 학습 내용

### 1. 자바 개발 환경 (Environment)
* **JRE (Java Runtime Environment):** 프로그램 **실행** 시 필요
* **JDK (Java Development Kit):** 프로그램 **개발** 시 필요 (JRE 포함)
* **Java Editions:**
    * **SE (Standard):** 기본/알고리즘용
    * **EE (Enterprise):** 대규모 서버용 (Servlet, JSP 등), 24시간 운영 환경에 적합
    * **ME (Micro):** 임베디드용 (현재 거의 미사용)

### 2. 실행 방식 및 특징
* **컴파일 & 인터프리터:** 자바는 두 방식을 모두 사용하는 하이브리드 구조
* **바이트코드 (.class):** "Write Once, Run Everywhere" (한 번 작성하면 어디서든 실행)

### 3. 컴퓨터 구조와 변수 (Computer Architecture)
* **폰 노이만 구조:** 입력 → 저장 → 출력 (연산 포함)의 흐름
* **버퍼 (Buffer):** 데이터 전송 시 임시 저장 공간 (`in` / `out`)
* **변수 (Variable):** 재활용 가능한 메모리 영역 (데이터를 담는 그릇)

### 4. 데이터 타입 (Data Type)
* **정수:** `byte`, `short`, `int`(기본), `long`
* **실수:** `float`, `double`(기본)
* **논리:** `boolean` (true/false)
* **표기법:** * `PascalCase`: 클래스명
    * `camelCase`: 변수명, 메서드명
* **참고:** 자바는 기본적으로 `unsigned` 타입을 지원하지 않음 (모든 정수에 부호 존재)
